# Mobile-based Health Info Access

This is my CSP (Community Service Project) website designed to improve health awareness and accessibility for underserved communities.

## Features
- Daily health tips
- Emergency contacts
- Clinic and hospital location info
- Health record storage

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project aims to enhance quality of life by providing important health information through a mobile-friendly platform.
